# Prompt wykonawczy — streszczenie tekstu naukowego

Reference: {Fourcade and Healy, 2024 #118020}

Summarize the scientific text file in `*.md` format located in this directory.

Before you begin:

1. Read `AGENTS.md` in this directory—it contains the rules, output structure, and a sample summary.
2. Read the `*.md` file containing the scientific text to be summarized.

Then, perform the summary strictly according to the rules and template in `AGENTS.md`.

Remember: each paragraph must end with the reference tag mentioned above (`{AUTHOR, YEAR #ID}`, or using `@page` if `== page N ==` page markers are present).

Save the result as a new `*.md` file in this directory.
